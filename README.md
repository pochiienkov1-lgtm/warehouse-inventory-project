# Warehouse Inventory Project

Учебный консольный проект на Python для складского учёта товаров.

Проект позволяет:
- вводить товары через консоль;
- безопасно обрабатывать числовой ввод;
- рассчитывать общую стоимость товара;
- определять статус остатка;
- сохранять товары в CSV-файл;
- читать товары из CSV-файла;
- создавать отчёт по товарам с низким остатком;
- создавать отчёт по товарам для дозаказа.

## Структура проекта

## Файлы проекта

### main.py

Главный файл запуска программы.

Отвечает за основной сценарий:

1. ввод товаров;
2. обработку товаров;
3. сохранение товаров в CSV;
4. чтение товаров из CSV;
5. создание отчётов.

### input_utils.py

Модуль безопасного ввода данных.

Содержит функции:

- `get_int_input()`
- `get_float_input()`

Эти функции защищают программу от неправильного ввода чисел.

### product_utils.py

Модуль для работы с товарами.

Содержит функции:

- `create_product()`
- `process_product()`
- `calculate_product_value()`
- `get_stock_status()`
- `get_low_stock_products()`
- `get_order_products()`
- `print_products()`
- `print_order_products()`

### csv_utils.py

Модуль для работы с CSV-файлами.

Содержит функции:

- `save_products_to_csv()`
- `read_products_from_csv()`
- `save_low_stock_report()`
- `save_order_report()`

Также использует `pathlib`, чтобы CSV-файлы создавались в папке проекта независимо от папки запуска программы.

## Как запустить проект

Перейти в папку проекта:

```bash
cd C:\python_automation\python_modules
````

Запустить программу:

```bash

```

Или:

```bash

```
